# Иерархический просмотрщик подразделений

Простое десктопное приложение на Python + PyQt5 для отображения и редактирования иерархической структуры из SQLite.

**Что сделано:**
- Дерево на QTreeView с кастомной моделью (QAbstractItemModel)
- Отображение имени, иконок из BLOB и цвета по состоянию (state: 0=красный, 1=жёлтый, 2=зелёный)
- Редактирование имени
- Добавление дочерних элементов (наследуют state родителя)
- Удаление узлов (с каскадным удалением детей)
- Контекстное меню
- Интерфейс через Qt Designer (.ui файл)

**Архитектура:**
SQLAlchemy 2.0 → Service → TreeNode → QAbstractItemModel → QTreeView

Интерфейс минималистичный (только дерево), но весь дизайн легко дорабатывается в Qt Designer.

Запуск: `python main.py`
